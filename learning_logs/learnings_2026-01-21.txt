================================================================================
                     TOPIC 1: GOOGLE CLOUD PLATFORM SETUP
================================================================================

Setting up GCP for the Data Engineering Zoomcamp infrastructure.

--------------------------------------------------------------------------------
INSTALLING GOOGLE CLOUD SDK
--------------------------------------------------------------------------------

# Step 1: Check system architecture
uname -s                    # Returns: Linux
uname -a                    # Full system info

# Step 2: Download gcloud CLI for Linux
curl -O https://dl.google.com/dl/cloudsdk/channels/rapid/downloads/google-cloud-cli-linux-x86_64.tar.gz

# Step 3: Extract the archive
tar -xf google-cloud-cli-linux-x86_64.tar.gz

# Step 4: Run the installer
./google-cloud-sdk/install.sh

# Step 5: Initialize gcloud (opens browser for auth)
gcloud init

--------------------------------------------------------------------------------
GCLOUD CLI COMMANDS
--------------------------------------------------------------------------------

| Command                              | Purpose                              |
|--------------------------------------|--------------------------------------|
| gcloud init                          | Initialize and configure gcloud      |
| gcloud auth list                     | List authenticated accounts          |
| gcloud auth login                    | Login with Google account            |
| gcloud auth application-default login| Set up application default creds     |
| gcloud config list                   | Show current configuration           |
| gcloud projects list                 | List all accessible projects         |
| gcloud config set project <ID>       | Set default project                  |

Service accounts are used for programmatic access to GCP resources.

--------------------------------------------------------------------------------
CREATING SERVICE ACCOUNT (via GCP Console)
--------------------------------------------------------------------------------

4. Grant roles:

   - Storage Admin
